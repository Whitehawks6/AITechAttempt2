# AITechExplained.com

Educate 2.5 billion non-tech people on AI (1995-2035) with only citable facts. No hype, no paywalls, no sales.

## Tech Stack

- Next.js 15 (App Router)
- Tailwind CSS (dark outer-space theme)
- TypeScript
- React Server Components
- Vercel (free tier)

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

This project is configured for one-click deployment on Vercel. Simply connect your repository to Vercel and deploy.

## Project Structure

- `/app` - Next.js App Router pages
- `/components` - React components
- `/public` - Static assets including timeline.json
- `/docs` - Project documentation
